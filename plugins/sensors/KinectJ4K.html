# KinectJ4K

Component Type: Sensor (Subcategory: Standard Input Devices)

The Kinect component interfaces to the Microsoft Kinect v1 camera over the J4K library. It provides the skeleton data of exactly one person in front of the kinect v1 camera.

For the installation the SDK from Microsoft is needed. The plugin was tested with the SDK-Version 1.5 (see here).

KinectJ4K plugin

# Requirements

  • A Microsoft Kinect v1 must be connected to the system.
  • The drivers from Microsoft (usually installed with the SDK) are required.
